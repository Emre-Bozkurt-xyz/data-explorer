# Data Explorer

Small full-stack web app for exploring tabular scientific datasets.

## Tech

- Backend: FastAPI, SQLAlchemy, Postgres
- Frontend: React, Vite, TypeScript, RTK Query, MUI
- Infra: Docker, docker-compose, GitHub Actions CI

## Features

- List datasets with search + pagination
- Dataset detail view with inferred schema (field types, null fraction, distinct counts)
- Record browser with pagination and basic search
- Bookmark records with per-user notes
